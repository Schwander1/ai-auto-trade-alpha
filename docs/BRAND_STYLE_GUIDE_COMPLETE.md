# Alpine Analytics LLC - Complete Brand Style Guide

## Brand Identity

**Company:** Alpine Analytics LLC  
**Tagline:** Adaptive AI Trading Signals  
**Short Tagline:** Provably.  
**Trademark:** Alpine Analytics®

## Logo System

### Primary Logo
- **Concept**: Geometric mountain peaks with embedded data signal line
- **Colors**: Neon gradient (cyan → pink → purple)
- **Use**: Main branding, headers, marketing materials
- **File**: `/brand/logo-primary.svg`

### Icon Logo
- **Size**: 64x64px optimized
- **Use**: Favicons, app icons, small spaces
- **File**: `/brand/logo-icon.svg`

### Wordmark
- **Font**: Orbitron (900 weight, 0.15em letter spacing)
- **Use**: Text-only applications
- **File**: `/brand/logo-wordmark.svg`

### Logo Usage Rules
- Always maintain minimum clear space (equal to logo height)
- Use on dark backgrounds only
- Never distort or rotate
- Always include trademark symbol (®) in formal documents

## Color System

### Primary Palette
- **Neon Cyan** `#18e0ff` - Primary accent, CTAs, links
- **Neon Pink** `#fe1c80` - Secondary accent, highlights  
- **Neon Purple** `#9600ff` - Tertiary accent, special features
- **Neon Orange** `#ff5f01` - Warnings, alerts, CTAs

### Backgrounds (Deep Blacks)
- **Pure Black** `#000000` - Deepest black
- **Primary** `#0a0a0f` - Main backgrounds
- **Secondary** `#0f0f1a` - Cards, surfaces
- **Tertiary** `#15151a` - Elevated surfaces
- **Border** `#1a1a2e` - Borders, dividers

### Gradients
- **Primary Gradient**: Cyan → Pink → Purple
- **Cyan-Pink**: Cyan → Pink
- **Pink-Purple**: Pink → Purple
- Use for hero text, buttons, accents

### Semantic Colors (Minority Use)
- **Success** `#00ff88` - Profits, success
- **Error** `#ff2d55` - Losses, errors
- **Warning** `#ff5f01` - Warnings
- **Info** `#18e0ff` - Information

## Typography

### Display Font: Orbitron
- **Use**: Hero headlines, major titles, logotype
- **Weight**: 900 (Black) for maximum impact
- **Letter Spacing**: 0.15em (2-3px) for futuristic look
- **Size**: 48px+ for hero, 36px+ for major headings

### Heading Font: Montserrat
- **Use**: Section headers, subheadings
- **Weight**: 700 (Bold) or 600 (Semibold)
- **Size**: 24-30px for H2, 20-24px for H3

### Body Font: Inter
- **Use**: Body text, UI elements, descriptions
- **Weight**: 400 (Regular) or 500 (Medium)
- **Size**: 16-18px minimum (desktop), 15-16px (mobile)
- **Line Height**: 1.6 for readability

### Mono Font: JetBrains Mono
- **Use**: Code, data, technical content
- **Weight**: 400 (Regular)
- **Size**: 14-16px

## Layout & Spacing

### Grid System
- **Columns**: 12-column responsive grid
- **Gap**: 24-32px between sections
- **Margins**: 24-32px for CTAs and info boxes

### Spacing Scale
- **XS**: 4px
- **SM**: 8px
- **MD**: 16px
- **LG**: 24px (preferred)
- **XL**: 32px (preferred for CTAs)
- **2XL**: 48px
- **3XL**: 64px

### Card Styling
- Background: `#0f0f1a` (secondary black)
- Border: 1px solid neon cyan with glow
- Padding: 24px (1.5rem)
- Shadow: Soft with neon-tinted glow
- Stats/metrics: Bold Orbitron with neon borders

## Effects & Animations

### Neon Glow
- **Cyan**: `0 0 20px rgba(24, 224, 255, 0.5), 0 0 40px rgba(24, 224, 255, 0.3)`
- **Pink**: `0 0 20px rgba(254, 28, 128, 0.5), 0 0 40px rgba(254, 28, 128, 0.3)`
- **Purple**: `0 0 20px rgba(150, 0, 255, 0.5), 0 0 40px rgba(150, 0, 255, 0.3)`
- **Gradient**: Combined glow for maximum impact

### Pulse Animation
- **Duration**: 2s
- **Timing**: ease-in-out
- **Use**: Buttons, live stats, changing metrics
- **Effect**: Gentle opacity and glow intensity change

### Hover Effects
- **Buttons**: Glow intensifies, slight lift (translateY -2px)
- **Cards**: Border glow increases
- **Links**: Underline with neon color

## Component Guidelines

### Buttons
- **Primary**: Neon cyan background, black text, glow on hover
- **Secondary**: Neon pink outline, transparent background
- **CTA**: Orange for urgency, gradient for special
- **Padding**: 16px vertical, 32px horizontal
- **Font**: Montserrat Bold, 16px

### Cards
- Background: `#0f0f1a`
- Border: 1px solid neon cyan (with glow)
- Padding: 24px
- Shadow: Soft with neon tint
- Stats: Orbitron font, large size, neon color

### Modals
- Background: `#0a0a0f` with backdrop blur
- Border: Neon cyan glow
- Close button: Neon pink
- Consistent spacing: 32px

### Tables
- White text on dark background
- Neon borders for important cells
- Minimal shading
- Clear hierarchy

## PDF/Report Styles

### Headers
- Black background (`#0a0a0f`)
- Large Orbitron headlines (neon cyan)
- Logo top left, date/time top right
- Neon line separator

### Sections
- Neon lines/borders to separate key sections
- Generous spacing (32px)
- Clear hierarchy with color coding

### Tables
- White text
- Minimal shading
- Neon borders for important data
- Clear alignment

## Social Media Assets

### Guidelines
- Keep gradients consistent with main palette
- Use bold hero text
- Include "Provably." tagline when appropriate
- Use icons, charts, or mini-mountains as visual anchors
- Maintain high contrast for readability

### Templates
- Instagram: 1080x1080px
- Twitter: 1200x675px
- LinkedIn: 1200x627px
- All with neon accents on black

## Accessibility

### Contrast Requirements
- **AA Standard**: Minimum 4.5:1 for normal text
- **AAA Standard**: Minimum 7:1 for normal text
- **Large Text**: Minimum 3:1 (AA) or 4.5:1 (AAA)

### Text Sizes
- **Minimum**: 16px for body text
- **Mobile**: 15-16px minimum
- **Desktop**: 16-18px preferred

### Focus States
- Clear, glowing feedback on interactive elements
- High contrast focus indicators
- Keyboard navigation support
