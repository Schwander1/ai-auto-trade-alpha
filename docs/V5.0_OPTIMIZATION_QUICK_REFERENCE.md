## üîß Configuration

### Backup Format
```python
# Parquet (default)
manager.create_backup(date, format='parquet')

# CSV (backwards compatible)
manager.create_backup(date, format='csv')
```

### Confidence Calibration
```python
from argo.ml.confidence_calibrator import ConfidenceCalibrator

calibrator = ConfidenceCalibrator()
calibrated = calibrator.calibrate(raw_confidence, symbol)
```

### Outcome Tracking
```python
from argo.tracking.outcome_tracker import OutcomeTracker

tracker = OutcomeTracker()
tracker.update_outcome(signal_id, exit_price, outcome='win')
stats = tracker.get_outcome_statistics(days=30)
```
