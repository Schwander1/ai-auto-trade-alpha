## Troubleshooting

### If Prop Firm Executor Stops

1. **Restart it:**
   ```bash
   ./scripts/start_prop_firm_executor_fixed.sh
   ```

2. **Check logs:**
   ```bash
   tail -50 /tmp/prop_firm_executor.log
   ```

3. **Verify port is available:**
   ```bash
   lsof -i :8001
   ```

### If Signals Not Executing

1. **Check confidence thresholds:**
   - Argo: 75% minimum
   - Prop Firm: 82% minimum

2. **Check market hours:**
   - Stocks: 9:30 AM - 4:00 PM ET
   - Crypto: 24/7
   - Enable 24/7 mode for continuous generation

3. **Check risk validation:**
   - Review executor logs
   - Check position limits
   - Verify daily loss limits
